# ethereum-store — React Playground (Vite)

This directory is a **playground** to develop, test, and document usage examples of `ethereum-store` with React + Vite. It enables fast iteration with HMR, validation of real flows, and sharing reproducible demos.

## Features

- Hot Module Replacement (HMR) with Vite.
- Happy‑path and error‑case examples.
- Direct integration with the `ethereum-store` source during development (no publishing required).

## Requirements

- Node ≥ 18
- npm

## Scripts

- `dev`: `vite`
- `build`: `vite build`
- `lint`: `eslint .`
- `preview`: `vite preview`

## Local development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the browser at the URL shown by Vite (defaults to http://localhost:5173).
